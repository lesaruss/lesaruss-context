## Writing Rules

- Never use em-dashes anywhere. Use comma, period, colon, or plain hyphen.
- Orange #F69820 on white FAILS ADA contrast (2.22:1). Never use orange text on light backgrounds.
- All visual output must pass WCAG 2 AA before delivery. Run the check silently. No exceptions.
- White backgrounds for all deliverables. Never dark mode for outputs.

## Brand Animation

Dot Pulse is a brand constant. All orange dots and periods pulse site-wide.
- 2.8s cycle, ease-in-out, opacity + radial glow
- Respects prefers-reduced-motion
