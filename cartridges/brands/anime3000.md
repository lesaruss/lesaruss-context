# Anime3000 Cartridge

## What This Is
Anime3000 was originally a commentary brand, now pivoting to a full anime production studio + community inside the LESARUSS Universe. Long-term vision: studio where fans vote on storylines, submit designs, co-create the shows. Shamanic Resin is the K-pop project tied into this universe (each anime episode debuts a song). Waiting room on April 20 soft launch.

## Domain
- Primary URL: `anime3000.com` (legacy brand domain)
- Production: Seedance 2.0 + Suno + Claude + Video Prompt Builder. See `playbooks/animation-playbook-2026-04-10.html`.

## Key People
- **Sean A. Russell** — founder, universe architect
- **Jasmine Laurent** — CCO, owns brand voice across animation properties
- **GM** — not assigned. Seat held, archive/showcase only, activation follows revenue signal (per `project_brand_gm_agents.md`)

## The Product
- Commentary era: past catalog (podcasts, written content) — feeds GeekFon Radio backlog when Captivate API lands
- Studio era: original anime shows built on Seedance 2.0 (video) + Suno (music). Each show gets an OP and ED theme song with accompanying music video.
- Community layer: fans vote on storylines, submit character designs, co-create

## Character Roster (Shamanic Resin-linked)
Momo, Bran, Vera, Aoi, Lex, Nilo Wave, Kingston. Music and character are inseparable: songs belong to episodes as releases, not as soundtracks. "Cat Dance" = Momo's cat ears episode, pairing locked.

## Overlap With Other Brands
- **Shamanic Resin** — inseparable sibling. Each anime episode debuts a Shamanic Resin song. Music is a character that lives inside the story world.
- **GeekFon Society** — music and animation production stack are shared. GeekFon Radio distributes Anime3000 music videos. Shares the Seedance + Suno waiting room music library.
- **Chester Is Cool** — shared production stack (different target audience). Chester = family/kids, Anime3000 = older audiences.
- **The Blueprint (sitcom)** — shared animation production stack. The Blueprint is the LESARUSS Universe anchor, Anime3000 is a genre-specific studio inside it.
- **Mad Tings** — separate music project, never conflate with Shamanic Resin. Mad Tings visits the fictional GeekFon Society in The Blueprint Ep 1.
- **TopSpot** — sibling archive/showcase brand from the pre-LESARUSS era.
- **LESARUSS Academy** — Academy's Creative Producer certification pipeline feeds Anime3000 production roles when the studio era activates.

## Rules
- Seedance 2.0 + Suno are the canonical production stack (TOS-safe, AI used as showcase, not shortcut)
- Character sheets and environment art must exist before video generation
- All prompts go through Claude + Video Prompt Builder skill
- File structure: `/animation/{character-sheets,environments,prompts,raw-clips,edited,social-cuts}/`
- Every show gets OP + ED theme songs (Suno) with accompanying music video (Seedance)
- All original music feeds the shared LESARUSS waiting room music library
- Music videos double as teasers/proof-of-concept for shows not yet fully funded
- Never conflate Anime3000 (the studio) with Shamanic Resin (the K-pop project that lives inside it)
- Never conflate Shamanic Resin with Mad Tings (different genre, different brand)
- Disclosure banner on every public surface (AI-and-fiction), members-first per Rule 4
- No em-dashes
