# LESARUSS 360 Cartridge

## What This Is
LESARUSS 360 is the modular personal operating system app inside the LESARUSS Universe. Each dimension of life = a separate module/cartridge members purchase individually or bundle. Health is the first built module. Trainer-shareable profile pages (clean, no internal ops visible). 360 is the personal app layer, The 111 Day Theory is the framework that lives inside it.

## Domain
- Primary URL: `360.lesaruss.ai` (subdomain convention locked 2026-04-22 for LESARUSS-prefixed brands)
- Stack: Next.js 16, Supabase, Vercel (shared LESARUSS platform)
- Mock: `LESARUSS Project/playbooks/lesaruss-360-health-profile-mock.html`

## Rules
- Trainer-shareable profile pages must be clean (no internal LESARUSS ops, no admin badges, no member-only widgets)
- Each module is a separate purchase; bundle logic lives in entry-engine-config.ts pattern
- Health data is sensitive: HIPAA-level care even if not technically regulated (B Corp commitment)
- Health module is Sean's live public case study; his data is intentionally visible, all other user data is private by default
- No em-dashes, WCAG 2 AA, members-first per `console/compliance.md` Rule 4
